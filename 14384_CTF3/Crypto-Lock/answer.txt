Analyzing the ransomware code revealed that it was using function time.time() for the key. So we need to find the time when the file was created. Did stat secret.docx.enc and converted the modify time to epoch time and replaced time.time() with it. Wrote a function decrypt(see decrypt.py) and called it on the ciphertext. This gives us the decrypted secret.docx file. Did scp and opened file on local machine to obtain the required flag.
